# Add back page with institutional logos at the bottom

Adds a final page with SMHI and HaV logos anchored to the page footer so
they always appear at the bottom regardless of content length.

## Usage

``` r
add_back_page(doc, cleanup)
```

## Arguments

- doc:

  An rdocx object.

- cleanup:

  Environment with a `files` character vector.

## Value

The modified rdocx object.
