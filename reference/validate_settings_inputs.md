# Validate settings form inputs

Validate settings form inputs

## Usage

``` r
validate_settings_inputs(url, ppm, n_taxa, n_imgs)
```

## Arguments

- url:

  Dashboard URL string.

- ppm:

  Pixels-per-micron numeric value.

- n_taxa:

  Number of mosaic taxa.

- n_imgs:

  Number of mosaic images.

## Value

Character error message or `NULL` if all valid.
